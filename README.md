### 1. Prerequisites
* **Python 3.10+**
* **NVIDIA GPU** + **CUDA Toolkit** (Required for the model to run)

### 2. Clone and Install
git clone https://github.com/seshu-green/Food_based_service.git
cd Food_based_service
pip install -r requirements.txt

### 3. Database Setup
Since `db.sqlite3` is included, you don't need to migrate. Just run:
python manage.py runserver

### 4. Access the Project
* **Website:** http://127.0.0.1:8000/
* **Login Page:** http://127.0.0.1:8000/user/log
* **Admin Panel:** http://127.0.0.1:8000/admin/
